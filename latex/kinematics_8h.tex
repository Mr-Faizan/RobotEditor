\doxysubsection{Detailed Description}
Declaration of \doxylink{class_kinematics}{Kinematics} classes for robot, joint, and tool kinematics calculations. 

This file defines the base class \doxylink{class_kinematics}{Kinematics} and its derived classes \doxylink{class_joint_kinematics}{Joint\+Kinematics}, \doxylink{class_robot_kinematics}{Robot\+Kinematics}, and \doxylink{class_tool_kinematics}{Tool\+Kinematics}. These classes provide an abstraction for calculating kinematic properties for robots, joints, and tools, using inheritance and polymorphism. \doxylink{class_joint_kinematics}{Joint\+Kinematics} includes structures for DH parameters and rotational values, with methods for setting and getting these properties.

Main features\+:
\begin{DoxyItemize}
\item Abstract base class for kinematics calculations
\item Derived classes for joint, robot, and tool kinematics
\item Structures for DH parameters and rotational values
\item Setters and getters for kinematic properties
\item Polymorphic interface for calculating DH parameters
\end{DoxyItemize}
