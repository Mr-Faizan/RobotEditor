\doxysubsection{Detailed Description}
Declaration of importvcmx class for importing and processing VCMX robot files. 

This file defines the importvcmx class, which provides methods to extract, convert, and parse VCMX robot files and their resources. It handles zip extraction, image conversion, resource parsing, and DH parameter calculation for robot simulation and modeling.

Main features\+:
\begin{DoxyItemize}
\item Extract VCMX files and convert 3DS models to OBJ format
\item Parse resource files (.rsc) and convert to JSON
\item Process kinematics, joint maps, offsets, geometry matrices, and robot variables
\item Run DH parameter calculator and validate results
\end{DoxyItemize}
