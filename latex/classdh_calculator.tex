\doxysubsection{Detailed Description}
Class for calculating and verifying DH parameters for robot joints. 

The \doxylink{classdh_calculator}{dh\+Calculator} class provides methods to read robot configuration from a JSON file, compute DH parameters for each joint, calculate transformation matrices, extract translation and rotation, and validate the results against geometry data.
