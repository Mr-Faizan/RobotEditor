\doxysubsection{Detailed Description}
Declaration of \doxylink{class_dynamics}{Dynamics} classes for robot, joint, and tool dynamics calculations. 

This file defines the base class \doxylink{class_dynamics}{Dynamics} and its derived classes \doxylink{class_joint_dynamics}{Joint\+Dynamics}, \doxylink{class_robot_dynamics}{Robot\+Dynamics}, and \doxylink{class_tool_dynamics}{Tool\+Dynamics}. These classes provide an abstraction for calculating dynamics properties for robots, joints, and tools, using inheritance and polymorphism. \doxylink{class_joint_dynamics}{Joint\+Dynamics} includes properties and methods for handling payload and kinematic parameters specific to robot joints.

Main features\+:
\begin{DoxyItemize}
\item Abstract base class for dynamics calculations
\item Derived classes for joint, robot, and tool dynamics
\item Setters and getters for joint dynamics properties
\item Polymorphic interface for calculating dynamics
\end{DoxyItemize}
