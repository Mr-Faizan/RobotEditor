\doxysubsection{Detailed Description}
Class representing a robot entity composed of joints. 

The \doxylink{class_robot}{Robot} class encapsulates all properties and behaviors of a robot, including its name, manufacturer, payload, reach, and a collection of joints. It supports adding, creating, and retrieving joints, and manages a unique ID for each robot.
