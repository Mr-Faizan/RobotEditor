Detailed Description

Declaration of dhCalculator class for DH parameter calculation and verification.

This file contains the declaration of the dhCalculator class, which provides methods to compute Denavit-Hartenberg (DH) parameters, transformation matrices, and verify DH parameters against robot geometry data. The class reads robot configuration from JSON files, processes kinematic and geometric data, and outputs results for robot modeling and validation.

Main features:

  • Parse robot configuration and kinematics from JSON files
  • Compute DH parameters for each joint
  • Calculate homogeneous transformation matrices
  • Extract translation and rotation from transformation matrices
  • Validate computed DH parameters against provided geometry matrices
  • Compare transformation matrices with a tolerance
