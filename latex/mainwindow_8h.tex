\doxysubsection{Detailed Description}
Declaration of the \doxylink{class_main_window}{Main\+Window} class for the Robot\+Editor application. 

This header defines the \doxylink{class_main_window}{Main\+Window} class, which provides the main user interface for Robot\+Editor. It manages robot configuration, 3D visualization, file operations, and user interaction via a tree view and context menus.

Main features\+:
\begin{DoxyItemize}
\item Create, import, open, save, and delete robot configurations.
\item Visualize robot models and joints in a 3D scene using Qt3D.
\item Edit robot and joint properties via a tree view.
\item Export and import robot data in .re package format.
\item Support for joint dynamics and kinematics editing.
\item Context menu actions for adding/removing joints and payloads.
\item JSON serialization and deserialization for robot data.
\end{DoxyItemize}
