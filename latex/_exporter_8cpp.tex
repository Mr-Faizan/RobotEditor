\doxysubsection{Detailed Description}
\doxylink{namespace_assimp}{Assimp} export interface. While it\textquotesingle{}s public interface bears many similarities to the import interface (in fact, it is largely symmetric), the internal implementations differs a lot. Exporters are considered stateless and are simple callbacks which we maintain in a global list along with their description strings.

Here we implement only the C++ interface (\doxylink{class_assimp_1_1_exporter}{Assimp\+::\+Exporter}).
