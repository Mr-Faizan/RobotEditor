\doxysubsection{Detailed Description}
Declaration of \doxylink{class_joint}{Joint} class for robot joint properties and dynamics. 

This file defines the \doxylink{class_joint}{Joint} class, which encapsulates properties, kinematics, dynamics, and visualization data for a robot joint. It provides methods for setting and getting joint parameters, managing kinematics and dynamics, and handling visualizations.

Main features\+:
\begin{DoxyItemize}
\item Store joint properties (name, range, speed, coefficients)
\item Manage kinematics and dynamics objects
\item Handle multiple visualizations
\item Provide setters and getters for all properties
\item Add, create, and retrieve dynamics by payload number
\end{DoxyItemize}
