\doxysubsection{Detailed Description}
Declaration of \doxylink{class_robot_lib}{Robot\+Lib} class for robot management, file I/O, and VCMX import. 

This file defines the \doxylink{class_robot_lib}{Robot\+Lib} class, which provides methods for managing a collection of robots, loading and saving robot data to JSON, importing robots from VCMX files, and handling robot packages.

Main features\+:
\begin{DoxyItemize}
\item Manage a collection of \doxylink{class_robot}{Robot} objects
\item Add, remove, and retrieve robots by name or ID
\item Load and save robot data from/to JSON files
\item Import robots from VCMX files and parse robot data
\item Zip and extract robot package folders
\item Print robot and joint data for debugging
