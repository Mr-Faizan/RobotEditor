57/* Use prefixed names for the symbols from stb_image as it is a very commonly embedded library.
58 Including vanilla stb_image symbols causes duplicate symbol problems if assimp is linked
59 statically together with another library or executable that also embeds stb_image.
60
61 Symbols are not prefixed if using Hunter because in such case there exists a single true
62 stb_image library on the system that is used by assimp and can be used by all the other
63 libraries and executables.
64
65 The list can be regenerated using the following:
66
67 cat "path/to/stb/stb_image.h" | fgrep STBIDEF | fgrep '(' | sed -E 's/\*|\‍(.+//g' | \
68 awk '{print "#define " $(NF) " assimp_" $(NF) }' | sort | uniq
69*/
