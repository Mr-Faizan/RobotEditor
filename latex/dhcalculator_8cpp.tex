Implementation of \doxylink{classdh_calculator}{dh\+Calculator} class for DH parameter calculation and verification. 

This file contains the implementation of the \doxylink{classdh_calculator}{dh\+Calculator} class, which provides methods to compute Denavit-\/\+Hartenberg (DH) parameters, transformation matrices, and verify DH parameters against robot geometry data. The class reads robot configuration from JSON files, processes kinematic and geometric data, and outputs results for robot modeling and validation.
