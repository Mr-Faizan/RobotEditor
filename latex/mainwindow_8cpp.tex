\doxysubsection{Detailed Description}
Implementation of all UI Implemenation for the Robot\+Editor application. 

This file contains the implementation of the \doxylink{class_main_window}{Main\+Window} class, which serves as the main user interface for the Robot\+Editor application. It provides functionalities for robot management, 3D visualization, file operations, and user interaction through a tree view and context menus.

Main features\+:
\begin{DoxyItemize}
\item Create, import, open, save, and delete robot configurations.
\item Visualize robot models and joints in a 3D scene using Qt3D.
\item Edit robot and joint properties via a tree view.
\item Export and import robot data in .re package format.
\item Support for joint dynamics and kinematics editing.
\item Context menu actions for adding/removing joints and payloads.
\item JSON serialization and deserialization for robot data.
\end{DoxyItemize}
